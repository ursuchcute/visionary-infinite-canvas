# 支持项目

如果本项目对你有帮助，可以通过以下方式支持持续维护：

- 给 [GitHub 仓库](https://github.com/ursuchcute/visionary-infinite-canvas) 点 Star。
- 提交清晰可复现的 Issue，帮助定位问题。
- 完善文档、测试或功能并提交 Pull Request。
- 在使用本 Fork 时保留上游来源、许可证和当前版本源码入口。

本页面当前不提供广告、付费授权或账号赞助入口，避免沿用上游维护者的个人联系方式。
