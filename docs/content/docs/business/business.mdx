# 服务与支持

如果你在使用 Visionary Infinite Canvas 时需要部署建议、功能讨论、模型渠道接入或问题排查，请通过本 Fork 的 GitHub 仓库联系维护者。

## 适合沟通的方向

- 自托管、内网部署和团队使用评估。
- 画布节点、生成流程、素材管理和插件扩展。
- OpenAI 兼容接口、模型渠道和提示词仓库接入。
- Bug、兼容性、安全性和文档改进。

## 联系方式

- 功能与问题：[GitHub Issues](https://github.com/ursuchcute/visionary-infinite-canvas/issues)
- 安全漏洞：[私密漏洞报告](https://github.com/ursuchcute/visionary-infinite-canvas/security/advisories/new)
